# Fichiers de fabrication et de conception de la carte Capteurs 
## Fichiers de fabrication
Dans ce répertoire on trouve les fichiers de fabrication qui peuvent être utilisés pour la réalisation des PCB de la carte processeur utilisée. Cette carte comporte le cablage du processeur et les connecteurs vers les différents éléments du capteur
L'aspect physique de la carte est donnée à la figure  ci-dessous 

![CarteCapteur](img/AQOR_Capteur.png)

et le schéma Kicad du PCB est donné ci-dessous

![Kicad_PCB_carte_capteur](img/Kicad_AQOR_Capteur_pcb.jpg)

## Fichiers de conception 
La conception a été réalisée avec KICAD 7.0 [https://www.kicad.org/](https://www.kicad.org/)
Les fichiers de conception sont donnés dans le fichier [Kicad_AQOR_BME80_INMP441_pro.zip](https://github.com/atena-aqor/aqor-s/blob/main/AQOR_Carte_Capteurs/Kicad_AQOR_BME80_INMP441_pro.zip)
